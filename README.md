##特性
* 支持最新版的[Egret Wing](http://developer.egret.com/cn/)进行开发
* 仅支持Egret Wing的格式
* 支持Egret Wing中大部分控件

## 快速入门
本项目目前没有发布正式版本，只有下载代码。

## 源码构建
项目使用gradle管理，直接运行`./gradlew build`即可。导入源码 运行即可
